Accessibility

Accessibility Statement

Last updated: July 13, 2026

Commitment

Orahood Custom Business Solutions aims to make this website usable for visitors with a range of devices, assistive technologies, and access needs. The site is built with semantic HTML, keyboard navigation, visible focus states, responsive layouts, and descriptive text for meaningful images.

Standards

The website is reviewed against WCAG principles: perceivable content, operable controls, understandable copy, and robust markup. OCBS treats accessibility as an ongoing quality requirement rather than a one-time checklist.

Known Limits

Some portfolio media includes detailed dashboard and application screenshots. OCBS provides descriptive alt text, captions, and transcript support so the core meaning is available without relying only on the image or video.

Feedback

If you have trouble using any part of the website, email dalton@orahood.net or call 405‑639‑8796. Include the page, what you were trying to do, and the access issue you encountered.
